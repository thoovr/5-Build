# 5-Build
Module 5 Build
The Indie Film Collective is multi-page webpage intended for local creatives interested in indie film about the organization.
The web pages included on the site are the home page (index.html), screenings page (screenings.html), workshops page (workshops.html), and collaborations page (collaborations.html).
The HTML is semantic and validated.
The CSS is responsive.
There is no Javascript.
URL is https://thoovr.github.io/5-Build
